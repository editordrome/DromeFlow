Sobre este Dashboard Educacional

Esta é uma réplica educacional em HTML puro do sistema DromeFlow. O objetivo é demonstrar visualmente como o layout e os componentes funcionam sem dependências de frameworks JavaScript.

Funcionalidades interativas:

  • Clique no botão de menu no header da sidebar para colapsar/expandir a barra lateral
  • Em dispositivos móveis, use o botão de menu no topo para abrir o menu lateral
  • Clique nos itens do menu para simular a navegação entre módulos
  • Os cards de métricas são clicáveis e mudam de cor ao serem selecionados
