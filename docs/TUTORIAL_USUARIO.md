## 1. Entrando no Sistema

Acesse o sistema pelo link fornecido pela sua organização.

Na tela de login:

1. Digite seu **e-mail** no campo indicado
2. Digite sua **senha**
3. Clique em **Entrar**

> 💡 O sistema lembra onde você parou. Na próxima vez que abrir, ele vai direto para o módulo que você estava usando.

---

## 2. A Tela Principal

Após o login, você verá duas áreas principais:

```
┌─────────────────────────────────────────────────┐
│  [≡] MENU LATERAL    │   ÁREA DE CONTEÚDO       │
│                      │                           │
│  🏠 Dashboard        │   (O módulo que você     │
│  📋 Dados            │    selecionou aparece     │
│  📅 Agenda           │    aqui)                  │
│  👥 Clientes         │                           │
│  ...outros módulos   │                           │
└─────────────────────────────────────────────────┘
```

**Menu lateral (à esquerda):** lista todos os módulos que você tem acesso. Clique em qualquer item para abrir aquele módulo.

> 💡 O menu fica recolhido por padrão para dar mais espaço à tela. Passe o mouse sobre ele para expandi-lo.

---

## 3. Escolhendo sua Unidade

No topo do menu lateral há um **seletor de unidade**. Se você tem acesso a mais de uma unidade (franquia, filial), clique nele para escolher qual unidade quer visualizar.

- Os dados de cada módulo mudam conforme a unidade selecionada
- Você pode selecionar **"Todas as Unidades"** para ver um resumo geral (disponível em alguns módulos)

## 4. 📊 Dashboard — Meu Painel

> **Para que serve:** Ver o desempenho da unidade em um só lugar — faturamento, número de atendimentos, clientes novos e repasse das profissionais.

### O que você vê ao entrar

No topo da tela há **4 cards** com os números do mês:

| Card | O que significa |
|---|---|
| **Faturamento** | Total de receita gerada no mês |
| **Atendimentos** | Quantidade de serviços realizados |
| **Clientes** | Quantas pessoas diferentes foram atendidas |
| **Repasse** | Total a ser pago às profissionais |

### Como trocar o período
No canto superior direito há um seletor de **mês/ano**. Clique nele e escolha o período que quer visualizar.

### Gráfico de barras / linhas
Abaixo dos cards há um gráfico mostrando a evolução mês a mês ao longo do ano. Você pode ver os dados por ano atual ou pelos últimos 12 meses.

### Clicando nos cards — submétricas
Cada card é clicável! Ao clicar em um card, o gráfico muda para mostrar detalhes sobre aquela métrica. Por exemplo:

- **Faturamento** → você pode ver o **Ticket Médio** (valor médio por serviço)
- **Atendimentos** → você pode ver quantos foram agendados antes e durante o mês
- **Clientes** → você pode ver quantos são **recorrentes** e quantos **não voltaram**
- **Repasse** → você pode ver a média por profissional

### Análise por Dia da Semana
Role a tela para baixo e veja quais dias da semana têm mais atendimentos na sua unidade — útil para planejar escala e divulgação.

## 5. 📋 Dados — Importar Atendimentos

> **Para que serve:** Importar a planilha de atendimentos (arquivo Excel) e consultar os registros já existentes.

### Visualizando os atendimentos
Ao entrar no módulo Dados, você verá uma tabela com todos os atendimentos registrados. Use o **filtro de período** (mês/ano) para ver um período específico.

- Clique no cabeçalho das colunas para ordenar
- Use a busca para encontrar um cliente ou profissional específica
- **Duplo clique em uma linha** abre todos os detalhes daquele atendimento

### Como importar uma planilha

1. Clique no botão **"Importar"** (ícone de upload)
2. Selecione o arquivo **Excel (.xlsx)** no seu computador
3. Aguarde o processamento — uma barra de progresso aparecerá
4. Ao finalizar, os novos atendimentos aparecerão na tabela

> ⚠️ **Atenção:** O sistema é inteligente. Se você importar um arquivo com atendimentos que já existem, ele **atualiza** os existentes em vez de duplicar. Só serão removidos registros que não estão mais na nova planilha.

### Exportar os dados
Clique no botão **"Exportar"** para baixar os dados filtrados em formato Excel.

## 6. 📅 Agenda — Escala de Profissionais

> **Para que serve:** Organizar a escala do dia — ver quais profissionais estão disponíveis e atribuí-las aos atendimentos.

### Visão geral da tela

A tela da Agenda é dividida em 3 colunas:

```
┌──────────────────┬────────────────┬────────────────┐
│   CALENDÁRIO     │  PROFISSIONAIS │  ATENDIMENTOS  │
│                  │    LIVRES      │    DO DIA      │
│  [calendário]    │                │                │
│                  │  • Ana Lima    │  ○ Cliente A   │
│  [métricas da    │  • Maria Silva │  ○ Cliente B   │
│   semana]        │  • João Souza  │  ○ Cliente C   │
└──────────────────┴────────────────┴────────────────┘
```

### Como usar o Drag & Drop (arrastar e soltar)

1. No calendário, clique no **dia** que quer organizar
2. Na coluna **"Profissionais Livres"** aparecem as profissionais disponíveis naquele dia
3. Na coluna **"Atendimentos"** aparecem os serviços sem profissional atribuída
4. **Arraste** o nome de uma profissional e **solte** sobre um atendimento para fazer a atribuição
5. O sistema salva automaticamente

> 💡 Profissionais com disponibilidade de 4h ou 6h não podem ser atribuídas a atendimentos de 8h — o sistema bloqueia essa ação.

### Abas da Agenda

**Agenda Principal:** organiza o dia via arrastar e soltar (explicado acima).

**Gestão:** tabela com a disponibilidade de todas as profissionais na semana/mês. Você vê quem disse SIM ou NÃO para cada dia. Se uma profissional disse NÃO mas tem atendimento, aparece o aviso **"ATENÇÃO!"** — clique nele para ver os detalhes do conflito.

**Configurações:** define quais dias do mês estão abertos para as profissionais informarem disponibilidade. Após configurar, clique em **Salvar**. O link da agenda pública é gerado automaticamente para ser enviado às profissionais.

### Como a profissional informa disponibilidade?
Ela acessa um link gerado pelo sistema (você encontra nas Configurações da Agenda). Pelo celular dela, ela:
1. Informa o WhatsApp
2. Vê os dias disponíveis
3. Marca SIM ou NÃO para cada dia
4. Clica em Enviar

As respostas aparecem automaticamente na aba **Gestão** em tempo real.

## 7. 🗓️ Agendamentos — Ver o Dia de Hoje

> **Para que serve:** Ver rapidamente todos os atendimentos de um dia específico com informações completas.

### Como usar

1. Use o calendário para selecionar o **dia** que quer ver
2. A lista abaixo mostra todos os atendimentos daquele dia com:
   - Nome do cliente
   - Nome da profissional
   - Horário
   - Endereço
   - Status atual

> 💡 Esta tela atualiza **automaticamente** — se um atendimento novo for cadastrado, ele aparece sem precisar recarregar a página.

### Disparar Webhook
Se sua unidade está integrada com automações, há um botão para **disparar um webhook** manualmente. Isso é útil para acionar lembretes ou notificações externas para os clientes do dia.

## 8. 👥 Clientes — Quem Voltou e Quem Sumiu

> **Para que serve:** Entender o comportamento dos seus clientes — quem está voltando, quem é novo e quem parou de ir.

### Os 4 indicadores

Ao topo da tela você encontra 4 números:

| Indicador | O que significa |
|---|---|
| **Total** | Quantos clientes foram atendidos neste mês |
| **Recorrentes** | Clientes que vieram este mês E no mês anterior |
| **Novos** | Clientes que vieram este mês pela primeira vez (ou depois de um longo período) |
| **Atenção ⚠️** | Clientes que vieram no mês passado mas **não voltaram** este mês |

### A lista de clientes

A tabela mostra cada cliente com:
- Nome
- Tipo de serviço (Comercial ou Residencial)
- Último atendimento
- Quantitas vezes veio nos últimos 3 meses (M, M-1, M-2)

### Como ver os clientes em atenção
Clique no card **"Atenção"** — a tabela filtra automaticamente para mostrar apenas os clientes que não retornaram. Use essa lista para ações de retenção (ligação, mensagem, oferta especial).

### Ver o histórico de um cliente
Dê **duplo clique** em qualquer linha da tabela para abrir o histórico completo daquele cliente — todos os atendimentos, datas e valores.

## 9. 📒 Base de Clientes

> **Para que serve:** Consultar qualquer cliente que já foi atendido, independente do período.

Funciona como uma **agenda de clientes**. Diferente do módulo "Clientes" (que mostra só quem veio no mês), aqui você encontra todos que já passaram pela unidade.

Use a busca para encontrar um cliente pelo nome e veja seu histórico completo com duplo clique.

## 10. 💼 Comercial — Acompanhar Leads

> **Para que serve:** Gerenciar as propostas e negociações da sua equipe comercial em formato de quadro visual (Kanban).

### Como funciona

A tela mostra um **quadro com colunas**, onde cada coluna representa uma etapa da venda:

```
 LEADS → ANDAMENTO → GANHOS → PERDIDOS → AGUARDANDO
  [card]   [card]     [card]             [card]
  [card]   [card]              [card]
```

Cada **card** representa um lead (possível cliente ou contrato).

### O que fazer

- **Mover um card:** arraste-o de uma coluna para outra conforme o avanço da negociação
- **Ver detalhes:** dê duplo clique no card para abrir todas as informações e editar
- **Adicionar novo lead:** clique no botão de adicionar na coluna desejada

> 💡 Quando um lead vai para a coluna **"Ganhos"**, o sistema automaticamente adiciona esse cliente ao cadastro da unidade.

## 11. 📞 Pós-Vendas — Contato Após o Atendimento

> **Para que serve:** Registrar o contato feito com o cliente depois do atendimento — se ele ficou satisfeito, se quer reagendar, e qual foi o resultado.

### Como funciona

Após cada atendimento, o sistema cria automaticamente um registro de pós-venda com status **"Pendente"**. Sua tarefa é:

1. Entrar no módulo Pós-Vendas
2. Ver os registros **pendentes**
3. Ligar ou enviar mensagem para o cliente
4. Clicar no registro e registrar o resultado:
   - Contatado
   - Reagendou
   - Não atendeu
   - Outros

### Filtros rápidos
Use os botões de filtro no topo para ver apenas os registros de um status específico (ex: só os Pendentes ou só os que Reagendaram).

## 12. 🧩 Recrutadora — Seleção de Profissionais

> **Para que serve:** Acompanhar o processo de seleção de novas profissionais desde o primeiro contato até a contratação.

### O Quadro de Seleção

A tela mostra um **quadro visual** com as candidatas organizadas por etapa:

```
QUALIFICADAS → CONTATO → ENVIO DOC → TRUORA → TREINAMENTO → FINALIZADO
```

Também há colunas para **Não Aprovadas** e **Desistentes**.

### O que fazer

- **Mover uma candidata:** arraste o card dela para a próxima etapa
- **Ver detalhes:** clique no card para abrir as informações e anotar observações
- **Adicionar candidata:** clique no botão "+" na coluna "Qualificadas"

### Contadores rápidos
No topo da tela você vê quantas candidatas foram movimentadas **hoje**, nesta **semana** e neste **mês** — útil para acompanhar a produtividade da equipe de recrutamento.

## 13. 👩‍💼 Profissionais — Cadastro da Equipe

> **Para que serve:** Manter o cadastro atualizado de todas as profissionais da unidade — ativas e inativas.

### Navegando pela lista

Ao entrar, você vê uma lista com todas as profissionais. Use as abas no topo para filtrar:
- **Todas** — mostra todas as profissionais
- **Ativas** — apenas quem está trabalhando atualmente
- **Inativas** — quem foi desligada ou está de licença

Use a **barra de busca** para encontrar pelo nome ou número de WhatsApp.

### Ativar ou desativar uma profissional
Na lista, há um **botão de toggle** (chave liga/desliga) em cada linha. Clique nele para ativar ou desativar sem precisar abrir o cadastro completo.

### Ver o cadastro completo
Dê **duplo clique** em qualquer linha para abrir o cadastro. Você verá 3 abas:

**Início:** Informações principais — nome, WhatsApp, tipo de serviço (comercial/residencial), habilidades  
**Dados:** Documentos pessoais — RG, CPF, data de nascimento, estado civil, endereço  
**Histórico:** Todos os atendimentos desta profissional, com filtro por mês, e a **avaliação média** que recebeu nos pós-vendas

### Cadastrar nova profissional
Clique no botão **"Novo Cadastro"** no canto superior direito. Preencha os dados obrigatórios (nome e WhatsApp) e salve.

## 14. 📈 Prestadoras — Desempenho da Equipe

> **Para que serve:** Ver o desempenho geral da equipe de profissionais e do recrutamento em formato de painel gerencial.

### O que você encontra

**Resumo em 3 cards:**
- Profissionais ativas
- Cadastros no recrutamento (mês atual)
- Atendimentos realizados (mês atual)

**Painel de Profissionais:**
Uma tabela com o **ranking** das profissionais — você pode ordenar por quem fez mais atendimentos ou por quem gerou mais em repasse. Clique em qualquer profissional para ver todos os atendimentos dela no mês.

**Painel de Recrutadora:**
Métricas do mês: quantas foram cadastradas, qualificadas, reprovadas, desistentes, e quantas foram ativadas como profissionais neste mês.

### Trocar entre painéis
Use os botões no topo para alternar entre ver o painel de **Profissionais** ou o painel de **Recrutadora**.

## 15. 💰 Financeiro

> **Para que serve:** Registrar e acompanhar as receitas e despesas da unidade.

### O que fazer

- **Lançar uma entrada** (receita): clique em "Nova Receita", preencha o valor, data e categoria
- **Lançar uma saída** (despesa): clique em "Nova Despesa", preencha os dados
- **Visualizar**: a tela exibe um resumo dos lançamentos do período selecionado

### Categorias
Você pode personalizar as categorias de receita e despesa conforme as necessidades da unidade nas **Configurações Financeiras**.

## 16. 💬 Umbler — WhatsApp da Unidade

> **Para que serve:** Gerenciar as configurações do WhatsApp Business da unidade integrado ao Umbler Talk.

### Abas disponíveis

**Configurações:** Informações da integração (Organization ID, chaves de acesso). Normalmente configurado uma única vez pelo administrador.

**Bots:** Lista dos bots configurados para atendimento automático. Você pode ver e editar os bots existentes.

**Tags:** Etiquetas usadas para classificar os contatos no WhatsApp.

**Campos:** Campos extras que sua equipe preenche nos contatos do Umbler.

**Respostas Rápidas:** Templates de mensagens prontas que a equipe pode usar no atendimento. Você pode:
- Ver as mensagens cadastradas (coluna da esquerda)
- Criar ou editar mensagens (coluna da direita)
- Usar variáveis como `{{nome_cliente}}` para personalizar automaticamente

> 💡 Clique em uma resposta rápida na lista da esquerda para editá-la na direita.

## 17. 📖 Tutoriais — Central de Ajuda

> **Para que serve:** Encontrar manuais de uso, guias passo a passo e tirar dúvidas sobre cada funcionalidade do sistema.

### Como navegar

1.  A Central de Ajuda (Tutoriais) está no menu lateral como **Tutoriais** ou **Ajuda**.
2.  Na **lateral esquerda**, você verá uma lista de categorias (módulos).
3.  Clique em uma categoria para ver os tutoriais disponíveis.
4.  Se uma categoria tiver vários manuais, eles aparecerão logo abaixo do nome do módulo no menu lateral.

### O que você encontra em cada tutorial

-   **Texto explicativo:** Passo a passo detalhado.
-   **Imagens e Vídeos:** Capturas de tela para facilitar o entendimento.
-   **Formatação clara:** Destaques para pontos importantes e dicas.

> 💡 Se você for um Administrador, verá um botão **"CONFIGURAR TUTORIAIS"** no rodapé da lateral esquerda. Clique lá para gerenciar os conteúdos, trocar a ordem de exibição ou criar novas categorias.

## Perguntas Frequentes

### O sistema não está mostrando meus dados. O que faço?
Verifique se a **unidade correta** está selecionada no topo do menu lateral. Depois confirme se o **período** está correto no filtro de mês/ano.

### Fiz o upload da planilha mas os dados não apareceram corretamente.
- Certifique-se de que o arquivo é um Excel (.xlsx)
- Verifique se as colunas da planilha estão no formato correto
- Se o problema persistir, entre em contato com o administrador do sistema

### Perdi o link da Agenda para enviar às profissionais.
Entre no módulo **Agenda** → aba **Configurações**. O link ativo estará disponível lá.

### Como sei se uma profissional informou a disponibilidade?
Vá em **Agenda** → aba **Gestão**. Você verá uma tabela com SIM/NÃO para cada profissional/dia. Profissionais que ainda não responderam aparecem sem resposta.

### Minha senha não funciona.
Entre em contato com o administrador da sua organização para redefinir o acesso.

### Posso usar o sistema pelo celular?
Sim! O DromeFlow funciona pelo navegador do celular (Chrome, Safari). Para uma melhor experiência, acesse pelo computador ou tablet.

### O que significa o símbolo ⚠️ "ATENÇÃO" na agenda?
Significa que uma profissional marcou que **não está disponível** naquele dia, mas existe um atendimento atribuído a ela. Clique no aviso para ver os detalhes e resolver o conflito.

### Posso ver dados de todas as unidades ao mesmo tempo?
Sim, selecione **"Todas as Unidades"** no seletor de unidade. Nem todos os módulos suportam essa visão — os que suportam vão agregar os dados automaticamente.
